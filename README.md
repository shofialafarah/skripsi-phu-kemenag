## üìù Deskripsi Proyek
Proyek ini merupakan sistem informasi berbasis web yang dirancang untuk mengelola dan memfasilitasi pelayanan Ibadah Haji pada Kantor Kementerian Agama Kabupaten Banjar. Fokus utama sistem ini adalah efisiensi administrasi dan kemudahan akses informasi bagi calon jemaah.

## üöÄ Fitur Utama
* **Manajemen Data Jamaah:** Pengolahan data calon jemaah haji secara terpusat.
* **Manajemen Pelayanan:** Pendaftaran, Cek Estimasi Keberangkatan, Pembatalan, dan Pelimpahan Haji.
* **Monitoring Status:** Fitur pelacakan status administrasi keberangkatan jemaah.
* **Laporan Pelayanan:** Penarikan data laporan secara otomatis untuk keperluan kantor.
